Each design document is derived from an approved spec in `specs/`.

## How to Write a Design Doc

1. Copy `TEMPLATE.md` to a new file (e.g., `user-auth-design.md`).
2. Link back to the originating spec.
3. Fill in all sections before implementation begins.
4. Set status to `Draft`, then move to `Review`, then `Approved`.
